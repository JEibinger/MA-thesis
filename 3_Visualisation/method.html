Data analysis and visualisation

As a first test of method, data regarding placement of long-s in the manuscripts will be transferred from the original TEI-XML file to a CSV list via XSLT and further processed using Python, e.g. pandas.

Visualisations of data resulting from the semantic annotations can be done using already prepared data. The resulting visualisations can be used as mentioned.

Visualisation types

There is a wide variety of possibilities regarding the visualisation of information. The range of charts is seemingly endless. It is therefore essential to find the right fit for each information type you try to visualise.
